# SwiftFund

SwiftFund is a fintech website landing page project.  
It contains the following pages:

- Home page  
- Features page  
- About page  
- Contact Us page  
- Login page  
- Signup page (with a password strength tracker implemented using JavaScript)

## Features

- Responsive design for different screen sizes  
- Modern look and clean layouts  
- Password strength tracker on the signup page  
- Convincing and structured content across all pages  

## Purpose

The project demonstrates how a fintech platform could present its services in a professional way while also including interactive elements like the password strength tracker, 
to enable users set secure passwords for their accounts.
It also has a functional Contact Us page where messages can be reviwed in real time.

## Technologies Used

- HTML  
- CSS  
- JavaScript
